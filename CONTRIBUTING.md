# Contributing to MailCue

Thank you for your interest in contributing to MailCue! We welcome bug reports, feature requests, documentation improvements, and pull requests.

## How to Contribute

Please read our detailed **[Development and Contributing Guide](https://olib-ai.github.io/mailcue/guides/development.html)** on our documentation website. It covers:

*   Prerequisites and local development setup for both frontend and backend.
*   Running tests and code linters (`ruff`, `mypy`, `eslint`, `tsc`).
*   Step-by-step instructions on submitting a Pull Request.
*   Development principles and code standards.

## Code of Conduct

We expect all contributors to adhere to our **[Code of Conduct](CODE_OF_CONDUCT.md)**. Please report any unacceptable behavior to [hello@olib.ai](mailto:hello@olib.ai).
